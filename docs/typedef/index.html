    public defaultConfig: Object source

    Default Phaser configuration, for more settings see https://photonstorm.github.io/phaser3-docs/global.html#GameConfig

    Properties:

    NameTypeAttributeDescription
    type number
    • optional
    • default: Phaser.AUTO

    Phaser.AUTO, Phaser.CANVAS, Phaser.HEADLESS or Phaser.WEBGL. AUTO picks WEBGL if available, otherwise CANVAS.
    width number | string
    • optional
    • default: 1024

    The width of the game, in game pixels
    height number | string
    • optional
    • default: 768

    The height of the game, in game pixels
    zoom number
    • optional
    • default: 1

    Simple scale applied to the game canvas. 2 is double size, 0.5 is half size, etc
    resolution number
    • optional
    • default: 1

    The size of each game pixel, in canvas pixels. Values larger than 1 are "high" resolution
    parent Object | string
    • optional
    • default: null

    The DOM element that will contain the game canvas, or its id. If null (the default) or if the named element doesn't exist, the game canvas is inserted directly into the document body
    canvas Object
    • optional
    • default: null

    Provide your own Canvas element for Phaser to use instead of creating one.
    canvasStyle string
    • optional
    • default: null

    CSS styles to apply to the game canvas instead of Phaser's default styles.
    CanvasRenderingContext2D Object
    • optional

    Provide your own Canvas Context for Phaser to use, instead of creating one.
    scene Object
    • optional
    • default: null

    A scene or scenes to add to the game. If several are given, the first is started; the remainder are started only if they have { active: true }.
    seed string[]
    • optional

    Seed for the random number generator
    title string
    • optional
    • default: ''

    The title of the game shown in the browser console
    url string
    • optional
    • default: 'http://phaser.io'

    The URL of the game shown in the browser console
    version string
    • optional
    • default: ''

    The version of the game shown in the browser console
    autoFocus boolean
    • optional
    • default: true

    Automatically call window.focus() when the game boots. Usually necessary to capture input events if the game is in a separate frame
    input boolean | InputConfig
    • optional

    Input configuration, or false to disable all game input. See https://photonstorm.github.io/phaser3-docs/global.html#InputConfig
    disableContextMenu boolean
    • optional
    • default: false

    Disable the browser's default 'contextmenu' event (usually triggered by a right-button mouse click)
    banner boolean | BannerConfig
    • optional
    • default: false

    Configuration for the banner printed in the browser console when the game starts. See https://photonstorm.github.io/phaser3-docs/global.html#BannerConfig
    dom DOMContainerConfig
    • optional

    The DOM Container configuration object, see https://photonstorm.github.io/phaser3-docs/global.html#DOMContainerConfig
    fps FPSConfig
    • optional

    Game loop configuration, see https://photonstorm.github.io/phaser3-docs/global.html#FPSConfig
    render RenderConfig
    • optional

    Game render configuration, see https://photonstorm.github.io/phaser3-docs/global.html#RenderConfig
